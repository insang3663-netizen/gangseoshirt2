# 강서 유흥 셔츠룸 - 럭셔리 프리미엄 웹사이트

순수 HTML/CSS/JavaScript로 만든 고성능 럭셔리 셔츠룸 웹사이트입니다.

## 특징

- ✨ 럭셔리 미니멀리즘 디자인
- 🎨 골드, 크림, 차콜 색상 팔레트
- 📱 완벽한 반응형 디자인
- ⚡ 초고속 로딩 (프레임워크 없음)
- 🔍 SEO 최적화
- 📍 위치, 연락처, 운영시간 정보

## 설치 및 실행

### 로컬 개발

```bash
npm install
npm run dev
```

### 빌드

```bash
npm run build
```

빌드된 파일은 `dist` 폴더에 생성됩니다.

## 배포

### Vercel 배포

1. GitHub에 푸시
2. Vercel에서 프로젝트 연동
3. 자동 배포 완료
